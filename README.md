## Introdução
Serão criadas as seguintes carteiras:

* Carteira Eficiente
* Carteira de Variância Mínima
* Carteira Equal Weight Equal Weight (Pesos Iguais)

Os ativos utilizados foram:
| Ativo | Setor                        |
| ----- | ---------------------------- |
| MOVI3 | Locação de veículos          |
| COGN3 | Educação                     |
| EZTC3 | Construção civil             |
| LOGG3 | Logística imobiliária        |
| FLRY3 | Saúde e medicina diagnóstica |

O perfil considerado foi o de um investidor Arrojado/Amante de risco, utilizando coeficiente de aversão ao risco igual a 0.5, indicando maior tolerância à volatilidade em busca de retornos superiores.


A análise foi fundamentada na Teoria do Portfólio de Harry Markowitz
